Send a GET request to /getToken with the parameter 'email' set

The following must be done for the service to run correctly

  • You must make sure a file named key.json is present with all the previous keys you found inserted to the path '/usr/src/app' of the container in the format {"email": "you@something.com", "key": ["KEY1", "KEY2", "KEY3", "KEY4"]}
