EcoTrail - Lets answer some simple questions


Usage

  • Do a HTTP POST to this address with this JSON payload
    {"email": "your@email.add", "key": "if you got one, write here. If this is your first question, leave blank"}
  • You will get a question like this
    {"result": "ok", "question" : "will you marry me?"}
  • Find the answer. Send in the answer with the JWT in the Authorization header as a bearer in this style
    {"answer":"I cannot. I am already married", "email": "your@email.add", "key": "if you got one, write here. If this is your first question, leave blank"}
  • Analyze the result and repeat answering the questions in the same manner till you get the key to the next step.
